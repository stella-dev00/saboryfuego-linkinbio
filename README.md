# Proyecto Link in Bio - Sabor y Fuego

Este proyecto contiene el código base (HTML y CSS) para una página estilo "Link in Bio" con un catálogo de productos integrado.
Está optimizado para visualización móvil y listo para ser desplegado en servicios de hosting estático como GitHub Pages o Netlify.

## Estructura de Archivos
- `index.html`: Estructura principal de la página, tarjetas de productos y enlaces de WhatsApp generados.
- `assets/css/style.css`: Estilos visuales adaptados a la marca (colores, tipografías y responsividad).
- `assets/img/`: Carpeta vacía lista para que subas las imágenes de los productos (idealmente formato cuadrado 1:1, ej. 800x800px).

## Cómo Desplegar en GitHub Pages
1. Inicia sesión en [GitHub](https://github.com/).
2. Crea un nuevo repositorio (ej. `saboryfuego-linkinbio`).
3. Sube todos los archivos de esta carpeta al repositorio.
4. Ve a la pestaña **Settings** del repositorio.
5. En el menú izquierdo, haz clic en **Pages**.
6. En "Source", selecciona la rama `main` (o `master`) y guarda.
7. ¡En un par de minutos tu página estará en vivo!

## Cómo Desplegar en Netlify
1. Crea una cuenta en [Netlify](https://www.netlify.com/).
2. En tu panel (Dashboard), haz clic en **Add new site** > **Deploy manually**.
3. Arrastra esta carpeta completa (`link_in_bio_project`) al recuadro.
4. Netlify te dará un enlace instantáneo (ej. `nombresitio.netlify.app`).
5. (Opcional) Puedes configurar un dominio personalizado desde la configuración del sitio.
